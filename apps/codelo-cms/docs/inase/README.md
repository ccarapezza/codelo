# Fuentes INASE

Copias de los documentos oficiales sobre los que se construyó `/semillas`.
Se versionan acá porque las URLs de `argentina.gob.ar` llevan número de versión
en el nombre del archivo (`v09-24`) y rotan sin aviso al reeditarse.

## `inase-categorias_rncyfs_v09-24.pdf`

Leyenda de las categorías del RNCyFS (A–P). Es la fuente de
`apps/codelo-web/lib/categorias-rncyfs.ts`.

- Descargado el 21/07/2026 de
  <https://www.argentina.gob.ar/sites/default/files/inase-categorias_rncyfs_v09-24.pdf>,
  enlazado desde el buscador oficial del padrón
  (<https://gestion.inase.gob.ar/empresas/empresas> → "Categorías - Referencias").
- Norma de respaldo: **Resolución INASE 474/2024, Anexo I** (B.O. 27/09/2024) —
  <https://servicios.infoleg.gob.ar/infolegInternet/anexos/400000-404999/404578/norma.htm>.
  Es la referencia estable; el PDF es su versión divulgativa.

Antes de tocar la tabla de categorías, releer ambos. No parafrasear de memoria:
ese texto le dice al público para qué está habilitado un vendedor de semillas.

### Dos cosas que el PDF no dice y conviene tener presentes

- **Las categorías describen la actividad, no la especie.** Un operador con `F`
  puede expender semilla de las especies para las que está inscripto, no de
  todas. El padrón no publica ese alcance.
- **Res. INASE 653/2023** habilitó para *Cannabis sativa* L. las categorías
  **A, B, E, F, H y K**.
